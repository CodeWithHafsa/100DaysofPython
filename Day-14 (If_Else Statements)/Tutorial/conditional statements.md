
# *Conditional Operators*:
* Greater than >
* Less than <
* Greater than equal to >=
* Less than equal to <=
* Equal to ==
* Not equal to !=

---
*Example: Conditional Operators*
* It returns boolean turn
```python
a = int(input("Enter your age: "))
print("Your age is: ", a)
print(a>18)
print(a<=18)
print(a==18)
print(a!=18)
```

Output:
```
Enter your age: 18
Your age is:  18
False
True
True
False
```

## Based on this, the conditional statements are further classified into following types:
- if
- if-else
- if-else-elif
- nested if-else-elif.
